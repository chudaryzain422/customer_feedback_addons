POS | Refund Order Authorization

This module restricts POS order refunds, allowing access only to authorized group members.

1. The current user is not included in the "Manage Refund Orders" group.
2. Tap Action (or the three dots in Community Edition) to unlock the next step.
3. Tap the Refund button to initiate the process.
4. Unfortunately, if you're not part of the Manage Refund Orders group, a popup message will notify you of restricted access.
